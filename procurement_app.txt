The Project Structure
procurement_app/
│
├── .env                      # Store your Gemini API key here securely
├── requirements.txt          # List of Python dependencies
├── app.py                    # The main Flask server and API routes
│
├── services/                 # Your core engine modules
│   ├── __init__.py
│   ├── ai_engine.py          # Gemini API connection & Pydantic models
│   ├── rules_engine.py       # The Accept/Reject/Correct logic
│   └── vision_engine.py      # The Pillow bounding-box drawing logic
│
├── static/                   # Public files accessible by the browser
│   └── processed_images/     # Where we save the images with bounding boxes
│
└── templates/                # Your HTML frontend
    └── index.html
